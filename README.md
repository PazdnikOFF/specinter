# Specinter

Сайт [specinter.ru](http://specinter.ru)

## Замечания

1. Каждый товар является, по сути, не товаров, а своеобразной категорией. Такой товар содержит «блоки» (название из админки), которые прикрепляются к товару. Собственно, эти самые блоки и являются конечными товарами, которые можно заказать на сайте.

2. Каждый товар может содержать «Дубли». Это простая запись, которая сообщает о том, что данная позиция встречается еще и стакам-то артикулом и с таким-то. Некоторые дубли можно привязать к товару. Тогда у этого дубля появляется карточка (карточка товара в привычном исполнении)
