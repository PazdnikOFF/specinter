# Specinter

Сайт [specinter.ru](http://specinter.ru)

## Замечания

1. Каждый товар является, по сути, не товаров, а своеобразной категорией (Как на [https://www.exist.ru](exist.ru)). Такой товар содержит «блоки» (название из админки), которые прикрепляются к товару. Собственно, эти самые блоки и являются конечными товарами, которые можно заказать на сайте. Таблица `it_b_catitem`

2. Каждый товар может содержать «Дубли». Это простая запись, которая сообщает о том, что данная позиция встречается еще и стакам-то артикулом и с таким-то. Некоторые дубли можно привязать к товару. Тогда у этого дубля появляется (должна) карточка (карточка товара в привычном исполнении). Таблица `it_b_aarts`

3. Каждый товар содержит «Список торговых предложений» (Вложенные блоки) из таблицы `it_b_variant`. Эти блоки выводятся на странице и являются, собственно, конечным товаров, которые возможно заказать. Имеют цену и артикул.

4. Файл `/inc/modules/catalog.php`. Метод `_showGood` отвечает за вывод данных карточки товара

5. В карточке товара (см. пункт 5), чтобы добавить массив в итератор для шаблона, каждый элемент массива должен быть экземпляром `stdClass`

6. Ссылка вида `/catalog/path/to/_aview_b1236/` отобразит товар с `id` **1236** из таблицы `it_b_ablock`

7. Шаблонизатор [http://ets.sourceforge.net/](ETS). [http://ets.sourceforge.net/ets.pdf](Документация)

## Структура базы данных

| `it_b_aarts`   | Содержит список вложенных блоков **(Дубли)**                                                     | Колонка `blockparent` определяет к какому товару привязан блок<br>Колонка `good_id_arts` указывает на `id` товара из `it_b_catitem`. Это привязка к карточке товара этого дубликата |

## Структура директорий

- inc
  - class
    - **listing.php** - Класс, при помощи которого осуществляется выборки из БД
  - modules
    - **catalog.php** - Вывод данных каталога (карточки товара)
    - **search.php** - Поиск на сайте
  - **var.php** - Данные для подключения к БД
- templates
  - misc
    - **head.html** - Тег <head> шаблона сайта
- js
  - **main.js** - Основной файл JavaScript
- css
  - **main.css** - Основные стили
