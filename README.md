# specinter

Репозиторий портала запчастей китайской спецтехники **ООО «СПЕЦИНТЕР»**.

## Структура

- **`legacy/`** — текущий сайт specinter.ru (выгрузка с хостинга) + дамп БД
  `specinter.dump.sql`. Самописная PHP‑CMS. Держим как справочник данных и бизнес‑логики.
  Код нового портала на нём **не строится**; из него мигрируются данные.
- **`new/`** — новый портал (в разработке). Docker, Next.js + FastAPI + Postgres.
  - `new/docs/01-analysis.md` — анализ текущего сайта и БД.
  - `new/docs/02-architecture.md` — архитектура и предложение по новому порталу.

## Цель нового портала

Автономный продающий портал с ИИ‑оператором: автоприём прайсов поставщиков с почты,
подбор по артикулу производителя, расчёт сроков доставки по ге(поставщик→склад→клиент),
оплата физлиц, **прямая** интеграция с 1С:УНФ и ЭДО Диадок, бот в Telegram/WhatsApp/MAX,
админка с метриками. UI — в традициях Apple.

См. `new/docs/` для деталей.
